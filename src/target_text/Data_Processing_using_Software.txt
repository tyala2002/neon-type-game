分析を行うために新たな情報を作り出すデータの処理には、表計算ソフトウェアがよく利用され、データの合計を求める積算雨量の計算などが簡単に行える。セルに入力された式をコピーする際に、参照するセルの位置が自動的に変更される参照方式を相対参照といい、コピー先にかかわらず同じセルを参照するには行番号や列番号の前に「$」を付けた絶対参照を使用する。表計算ソフトウェアには、加減乗除の四則演算以外にも、引数をもとに定義された計算を行うSUM関数やAVERAGE関数などの多様な関数が用意されており、条件によって異なる結果を返すIF関数などもある。収集したデータを表やグラフにまとめて視覚的に表現することを情報の可視化といい、棒グラフや折れ線グラフを使って特徴や傾向を把握しやすくする。グラフから情報を読み取る際には、全体的な傾向のほかに、周期性や大きく変化した部分などの特徴に注目する必要がある。また、データを小さい順の昇順や大きい順の降順で並べ替えることや、条件に合ったデータだけを抽出することで、目的の値の検索やデータの傾向把握を容易にすることができる。
